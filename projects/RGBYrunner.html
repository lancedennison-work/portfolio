RGBY Runner is an endless runner, where you run into obstacles of random colors, however you have the power to change the color of the world. When you change the color of the world, the matching color blocks blend into the background and disappear, allowing you to pass through them safely. The goal is to last as long as you can, gaining points for each wall you pass through.

This game started as a prototype for a game development experience course I took at UCSC and I liked the idea so much, I made the full game after the class ended. I was very interested in the idea of an endless runner where you can do more than simply trying to avoid obstacles. It becomes almost like a rhythm game like guitar hero; as you come up to each obstacle, you change the color to match the obstacle so you pass through and get points.
